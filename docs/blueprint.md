# **App Name**: ScrimSync

## Core Features:

- Availability Voting: Allow users to vote for their availability in 30-minute intervals between 4:30 PM and 9:30 PM.
- Vote Heatmap: Visualize the voting results as a heatmap, showing the most popular time slots.
- Schedule Management: Enable users to schedule training sessions and tournaments with dates and times.
- Schedule Overlay: Display scheduled training sessions/tournaments on the voting map so users can coordinate their votes more effectively.
- Discord Integration: Automatically post the voting results and availability information to a specified Discord channel, with a tool to help the app determine if additional details or clarification is necessary.
- Player Profile: Allow users to set their username and also players to type in their Favorite tank and also select their favorite in game roles-Tank destroyer/Medium Tank/Heavy Tank/Assaulter/Defender/Light tank is scout

## Style Guidelines:

- Primary color: Vibrant purple (#A020F0) to give an energetic feel related to gaming.
- Background color: Light gray (#E0E0E0), a desaturated version of purple, creating a clean backdrop.
- Accent color: Electric blue (#7DF9FF), an analogous color to the primary purple, providing highlights.
- Font: 'Space Grotesk', a proportional sans-serif for both headings and body.
- Use clear and simple icons to represent availability slots, training events, and tournament events.
- A clean, responsive layout that works well on both desktop and mobile devices, with clear sections for voting, scheduling, and results display.
- Subtle animations on voting map to indicate vote counts
